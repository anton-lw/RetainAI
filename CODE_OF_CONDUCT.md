# Code of Conduct

## Our Commitment

RetainAI contributors, maintainers, and participating organizations commit to making participation in this project respectful, safe, and constructive.

Because this project concerns vulnerable populations, we hold ourselves to a high standard of care in both technical work and community conduct.

## Expected Behavior

- Be respectful and technically honest.
- Prioritize clarity over point-scoring.
- Welcome good-faith questions and criticism.
- Flag ethical, privacy, or safeguarding risks early.
- Accept correction when evidence changes.

## Unacceptable Behavior

- harassment, threats, or intimidation
- discriminatory or demeaning language
- doxxing or disclosure of private information
- retaliating against people who raise concerns
- dismissing safeguarding, equity, or privacy concerns without serious consideration
- knowingly sharing beneficiary data outside approved channels

## Scope

This Code of Conduct applies to:

- repository discussions
- issue and pull-request discussions
- community communication spaces associated with the project
- project events, demos, and working sessions

## Enforcement

Maintainers may remove content, limit participation, or suspend contribution access when behavior violates this code or creates safety risk.

## Reporting

Report conduct concerns to Anton Waagaard at `hello@antons.site`, or use the
most private available maintainer route if a different public steward is later
appointed.

## Relationship to Product Safety

This Code of Conduct governs the contributor community. Product-facing safety and harmful-content expectations are documented separately in [docs/community-safety.md](docs/community-safety.md).
